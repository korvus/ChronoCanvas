• Basic use

    In native javascript: In the event handler of your choice, call chronoCanvas, and use the canvasTarget parameter for target the canvas node. It will use the querySelectorAll function. 

    window.onload = function(){
  chronoCanvas({
    canvasTarget: ".chronometer1"
  });
}

  • speed and twitch

    Portions parameter is the distance traveled between each canvas refresh. It's a numeric value between 0 and 1. If you set to 0.5, there will be only two move for a full turn. If 0.25, 4 moves. If you want one move for one second like on a watch, calculate it this way: 1/60 = 0.166666 (the number after point you want), like here.
    Frequency time in milisecond between each canvas refreshing. In this example, you got 1 second between each move. Then, the full turn take 1 minut, as the circle is divided in 60 portions. 

    window.onload = function(){
  chronoCanvas({
    canvasTarget: ".chronometer2",
    portions: 60,
    frequency: 1000
  });
}
